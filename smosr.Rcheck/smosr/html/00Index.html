A suite of tools to access, download and explore ESA-BEC Soil Moisture and Ocean Salinity (SMOS) data in R

Documentation for package ‘smosr’ version 0.1.0

  • DESCRIPTION file.

Help Pages

download_smos Download BEC-SMOS soil moisture data
extract_smos Extract BEC-SMOS soil moisture estimates for specific geographical locations
find_smos Find BEC-SMOS soil moisture data in Barcelona Expert Center (BEC) server
list_smos List the BEC-SMOS data files stored on a local computer
missing_smos Print the dates for which BEC-SMOS soil moisture data have not been found
plot_raster_smos Draw a raster image of BEC-SMOS soil moisture data
plot_temporal_smos Plot temporal series of BEC-SMOS soil moisture data
set_credentials Set credentials to access Barcelona Expert Center (BEC) server
